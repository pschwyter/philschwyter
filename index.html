Blog Entry #3

Friday, September 26th, 2014

This personal website will be my work-in-progress as I prepare for and complete the Bitmaker Labs Web Developer bootcamp. I'll hopefully be updating it regularly with blog posts and new features!

I'm sure this site would be so much better with a twitter feed or a comments section for all my eager readers.

The prep course Bitmaker Labs has set up for us seems like a pretty good start to becoming a Web Developer. As someone with essentially zero coding skills it's been very helpful so far in teaching the basics of the command line, Ruby and HTML/CSS. Having reached the half-way point in the prep course I'm beginning to realize the majority of my progress comes in actually DOING things rather than following some guide step-by-step.
